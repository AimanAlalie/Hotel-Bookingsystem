# Tech Stack

## Frontend

- React 18
- Next.js (App Router)
- Tailwind CSS for styling

## Backend

- Supabase (Backend-as-a-Service)
  - PostgreSQL database
  - Authentication (Supabase Auth)
  - Storage (Supabase Storage)
  - Real-time subscriptions

## Internationalization

- Language switcher for multi-language support
- English (LTR) and Arabic (RTL) with full layout transformation
- Cultural date and number formatting

## Infrastructure

- Vercel or similar for Next.js hosting
- Supabase hosted services
