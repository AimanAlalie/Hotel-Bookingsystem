# Product Roadmap

## Phase 1: MVP (Core Booking Flow)

### Guest Features
- Multi-branch search with availability checking
- Room type selection with photos and details
- Complete booking flow with cost breakdown
- User registration and login
- View and manage reservations (view, modify, cancel)
- Language switcher (English/Arabic with RTL support)

### Admin Features
- Basic dashboard with occupancy overview
- View and manage all bookings
- Create manual bookings (phone/walk-in)
- Room and inventory management
- Basic pricing configuration
- Staff account management

## Phase 2: Reviews + Messaging

### Guest Features
- Review and rating system for completed stays
- Direct messaging with hotel staff
- Email and SMS notifications
- Pre-arrival information delivery

### Admin Features
- Review moderation
- Communication logs
- Email template customization
- Notification management

## Phase 3: Advanced Admin & Analytics

### Analytics & Reporting
- Real-time occupancy rates across branches
- Revenue metrics and booking trends
- Exportable reports (PDF, Excel, CSV)
- Branch performance comparison

### Advanced Pricing
- Seasonal and dynamic pricing
- Discount code management
- Special offers and promotions

### Operations
- Maintenance scheduling
- Housekeeping status integration
- Activity logging and audit trails
- Advanced permission customization
